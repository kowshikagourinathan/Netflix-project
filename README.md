# Netflix Capstone Project

**Description:** Exploratory data analysis and a simple classification example using `netflix_titles.csv` (predicting whether an item is a Movie or TV Show using basic features).

**Files**
- `main.py` — loads data from `data/netflix_titles.csv`, preprocesses, trains a RandomForest classifier, and prints metrics.
- `requirements.txt` — Python dependencies.
- `data/` — place `netflix_titles.csv` here before running.

**Run**
```bash
pip install -r requirements.txt
python main.py
```
